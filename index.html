What QuantumCache Is

QuantumCache is a server-side query and fragment cache for WordPress. It runs inside PHP, reuses database query results and rendered fragments, and reduces the amount of work WordPress has to repeat on later requests. It is designed for requests that still reach the application layer, including standard WordPress pages, WooCommerce catalog pages, page-cache misses, and selected logged-in traffic when Pro is configured.

QuantumCache uses Redis when available and can fall back to its own MySQL-backed store when Redis is unavailable. It complements a page cache or CDN instead of replacing either layer.

QuantumCache is not a page cache — it does not store or serve complete HTTP responses. It is not a CDN — it does not operate at the network edge. It operates below both in the request stack, reducing the cost of PHP execution for requests that reach it.

Shared Foundation

Free and Pro share the same underlying cache engine. The difference is not a separate product line. Pro extends the same model for more complex WooCommerce and logged-in behavior.

Free includes debug headers, purge tools, and wp quantumcache status / wp quantumcache flush. Pro adds optional metrics and CDN-aware purge integrations.

Pro extends the same engine with WooCommerce-aware behavior, optional logged-in caching for allowlisted roles, Prometheus-compatible metrics, and CDN cache-tag integrations for Cloudflare, Fastly, and LiteSpeed.

Free for Public WordPress, Pro for More Complex Behavior

Free covers public WordPress acceleration. Pro extends the same core for WooCommerce-aware and controlled logged-in behavior.

Free

QuantumCache Free

  • Same shared query and fragment cache engine
  • Built for anonymous front-end WordPress traffic
  • Redis as preferred store with MySQL fallback
  • Debug headers, purge tools, and WP-CLI status / flush
  • WooCommerce pages can still benefit as general public content

No WooCommerce-specific key variation, bypass rules, product invalidation, or logged-in caching controls.

Pro

QuantumCache Pro

  • Everything in Free
  • WooCommerce-aware cache variation, archive TTLs, and fragment coverage
  • Bypass rules for cart, checkout, account, Woo AJAX, and session or cart cookies
  • Product-aware invalidation for catalog and related WooCommerce changes
  • Optional logged-in caching for allowlisted roles with a separate TTL
  • Prometheus metrics plus CDN tag / purge integrations

Pro is not "faster Free." It is the same engine extended for correctness, control, and operations on more complex WordPress and WooCommerce sites. Compare tiers →
